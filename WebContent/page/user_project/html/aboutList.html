关于我们

嘉实安装公司创立于1994年，隶属于上海嘉实（集团）有限公司，专注于机电设备安装工程领域，具有一级机电设备安装、一级消防设施安装、一级机电设备维修安装、一级制冷暖通空调维修安装专业承包资质。

嘉实安装立足大上海、聚焦长三角，延伸大中原.，业务覆盖皖、桂、宁、苏 、锡、郑等地区，和万科房产、金螳螂装饰、合景泰富、世贸、陆家嘴集团、金科地产、栖霞建设、交通大学等三十多家行业名流长期携手，至今已完成千余个安装项目。项目涉及工业设施、酒店娱乐，置业商务、康养医美，行政金融，居家住宅、文化小镇等众多领域，涵盖工业和民用建筑工程的通用设备安装、电力线路与电力设备安装、制冷工程安装、通风与空调安装、楼宇自控与智能化工程安装、各类消防安装工程、城市亮化工程专业承包。

二十余年来，安装公司参建的工程荣获 “鲁班奖”、“全国优质工程奖”、“白玉兰奖”、“扬子杯”、“申安杯”、“绿色安装”等400余项。其中苏州金鸡湖凯宾斯基大酒店、苏州工业园区档案大厦荣获中国建设鲁班奖；苏州工业园区置业广场工程荣获国家优质工程银质奖；苏州工业园区恒宇国际中心大楼荣获全国优质工程；陆家嘴软件园软件产业研发楼，荣获上海市建设工程结构优质安装奖；上海同济大学行政楼、上海大学宝山校区扩建工程等荣获上海市白玉兰工程奖；江阴丹芙春城5#机电工程、腾飞新苏工业坊机电工程等荣获上海市申安杯优质工程奖。

嘉实安装，将VDC/BIM技术和绿色施工有机结合，贯穿于机电安装全生命周期。先后与美国斯坦福大学、同济大学、交通大学等高校展开合作，通过不断的探索研究，挖掘和推动了BIM的独特价值。为推动智慧建造，嘉实人不断创新突破，自主研发嘉实智慧安装平台，借助云计算，利用iPAD端、PC端、手机端实现数据采集、管理、分析、应用全过程的信息化。

奋斗二十余载，砥砺前行。嘉实安装积累了丰富、独特的技术成果，仅安装信息化管理领域已获得BIM模型轻量化缓存、一键检索自动生成二维码，视角实时捕捉快速定位等十余项发明专利和二十余项著作权，取得多项省部级工法及QC成果。作为上海市安装协会及绿色安装中心理事单位，安装公司先后获得上海市绿色安装先进集体，全国先进施工企业。

不忘初心，荣耀承诺。嘉实安装始终坚持创新驱动，转型发展，大力实施“智慧建造”的发展战略，着力提升生产工业化、信息智能化、产品标准化、劳务产业化、管理精细化的能力，始终秉承“用心用爱 打造人们梦想家园，智慧制造 根植全球每个角落，嘉实英才 引领创造人生奇迹”的企业远景，努力把嘉实安装打造成为集机电设计、施工一体化、机电运维、BIM技术全程服务、安装信息化管理平台研发等机电全生命周期的国际一流机电安装企业
